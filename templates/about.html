How does this work?

Input a link to the front page of a warcraft logs report, hit submit, and wait for the results. The url of the log should look something like https://www.warcraftlogs.com/reports/h3pHANjvGmayDKXF. Rank Report uses the WCL API to get fight information, then webscrapes the rankings for each boss kill and processes the information using Python. Ranking information for each report that's been analyzed is then stored in Redis, so each report only gets scraped a single time.
