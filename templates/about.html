How does this work?

Input a link to the front page of a warcraft logs report, the url should look something like https://www.warcraftlogs.com/reports/h3pHANjvGmayDKXF. Rank report uses the WCL API to get fight information, then webscrapes the rankings for each boss kill and processes the information using Python.

This isn't working for me.

Make sure the link you're putting in is to the front page of the WCL report. You can also directly put in the sixteen character log ID. As an example, for the log https://www.warcraftlogs.com/reports/h3pHANjvGmayDKXF, the log id is h3pHANjvGmayDKXF.

Currently, Rank Report only supports reports that have advanced combat logging enabled. I'm working on expanding it to include all types of reports.

If you have an issue that's not covered by either of the above, send a copy of the WCL report you're trying to analyze to notputzing @ gmail.com. I can try to take a look and see what might be driving the error.
